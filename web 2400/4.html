  4. 랄로의 방송용어

누구야! - 롤 할 때 팀원이나 적팀이 못하면 걸쭉한 사투리 톤으로 내뱉는다.

너 나가 쯧! - 트수가 방송 도중 열받게 하면 블랙할 때 사용한다.

머선129 - 방송 중 어이없는 일을 겪으면 “이게 머선 일이고”라며 탄식하는 것에서 파생된 유행어이다

자 드가자~ - 한타를 열 때 사용한다. 트위치에 널리 퍼져 트위치 전용 밈이 되었으며 인터넷 전역에서 쓰인다.

마루쉐~ - 반려견인 마루를 부를 때 어이 마루쉐~ 라고 부른다.

서운하네.. - 마음대로 되지 않거나 욕을 먹었을 때 사용한다.

불만 없제? - 롤에서 상대의 스펠이 빠졌을 때나, 턴을 소모하면서 체력이 빠졌을 때, 그것을 확인하면 사용한다.

오소이새끼야 - 적이 쓰는 논타겟팅 스킬을 피할 때 사용한다. 느리다는 뜻의 일본어 오소이와 이새끼야를 붙인 적절한 개드립이다

숨도 못쉬겠제~ - 경기 초반 솔킬을 1~2회 따며 라인전에서 매우 유리할 때 사용한다.

네에~ - 도네이션으로 헛소리가 나올 때, 혹은 귀찮거나 무시할 때 사용한다.

수고하세요...! - 아군과 갱 호응시 적군의 난입으로 아군을 지켜줄 수 없을 때나, 적의 논타켓 스킬을 유유히 피하며 사용하며, 적의 막타를 스틸했을 때도 사용한다.

컽!/가버렷! - 라인전에서의 솔킬 혹은 적극적인 갱킹으로 주로 킬을 냈을 때 사용한다. 이후 '에어맨을 쓰러뜨릴 수 없어[47] '를 재생하며 줌인 줌아웃 카메라무빙으로 매드무비를 보여준다.

다음... 다음... - 캐리하는 상황에서 더블 킬 트리플 킬 등, 상대방을 하나 둘 무심하게 잡아낼 때 사용한다.

너무 훌륭하고! - 팀원이 좋은 플레이를 했을 때 사용한다.

~하는 건 나였고... - 주로 적을 잡을 수 있었는데 오히려 자신이 죽을 때 사용한다

어질어질하네..

무야호

랄로만.. - 말로만하고 행동은 그렇지 않을때 사용한다.

그... 없나?

응애! - 도네이션을 받았을 때 리액션으로 사용하기도 하고 그냥 갑자기 외칠 때도 있다.

~~어떻게 생각하시는지 - 여성 시청자와 두오를 할 때 아무거나 물어보면서 사용한다.

인생 - 인쉔으로 발음한다. 죽기 직전 삶을 포기하면서 사용한다

췌아! - 롤이나 gta 등을 할 때 점프하는 기술을 쓰면서 외치는 말.

에라이 씹년아 - 상대를 죽일 때, 상대 집 가는 걸 막을 때, 죽기 전 발악을 할 때 등 다양하게 사용한다.

(옹알이) - 무언가를 먹으면서 말할 때 옹알이를 한다.응애 나 아기 랄로

아님랄로 - 아님 말고~ 로 본인 발언에 대한 책임 회피 등을 할 때 사용한다.
